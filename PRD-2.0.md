# 蝦皮手續費計算機 2.0 開發

## 角色設定

你是一位精通 **Tailwind CSS v4.1** 與 **Vanilla JavaScript** 的資深前端工程師，擅長開發響應式、具備深色模式且效能優化的靜態網頁應用。

## 專案目標

開發「蝦皮手續費計算機 2.0」版本，將原有的單次計算工具升級為具備「賣場管理」功能的系統。所有資料必須儲存在瀏覽器的 `localStorage`，並採用 **2026 年新制費率** 進行計算 。

## 技術棧要求

* **樣式**：Tailwind CSS v4.1（使用最新語法與原生深色模式支援）。
* **架構**：多頁面靜態網頁（Multi-page App），適用於 GitHub Pages 部署。
* **儲存**：瀏覽器 `localStorage`（JSON 格式）。
* **圖標**：Bootstrap Icons 或 Lucide Icons。

## 核心邏輯與數據結構

1. 2026 費率邏輯 

* 
**金流手續費**：固定 2.5% 。


* 
**免運方案一**：加收 6% 。


* 
**免運方案二**：每筆固定加收 NT$60 。


* 
**成交手續費**：預設 6%，一般賣家單件上限 NT$35,000，商城賣家無上限 。


* 
**活動日加成**：一般賣家 +2%，商城賣家 +3%（若參加蝦幣回饋則不加收） 。


* 
**蝦幣回饋**：5% 方案 (1.5%)、10% 方案 (2.5%) 。


* 
**預購加成**：單品勾選「較長備貨」額外 +3% 。

## 頁面細節要求

### 1. 全站共同佈局 (Layout)

* **Header**：
* 左側：Logo + 「蝦皮手續費計算機」字樣。
* 右側：首頁、快速計算、賣場設定、商品管理（桌面版橫排，手機版漢堡選單）。
* 包含深色模式切換開關（預設跟隨系統）。


* **Footer**：維持 Bruce Wu 原作資訊、GitHub/IG/Threads 連結、CC BY-NC-ND 4.0 授權聲明與免責聲明。

### 2. 分頁功能描述

* **首頁 (`/`)**：產品功能介紹、改版重點、使用導覽入口。
* **快速計算 (`/calc`)**：
* 升級版 1.0 計算機，側邊欄（或選單）可快速選擇「商品管理」中的已存商品。
* 顯示詳細的摺疊式費用明細 。




* **賣場設定 (`/settings`)**：
* 設定全域參數：一般/商城、免運方案、蝦幣回饋、稅務設定（1% 稅籍/5% 發票/代開）。
* 匯入/匯出按鈕：匯出為 `.json` 檔案並呼叫手機原生 `Web Share API`。


* **商品管理 (`/management`)**：
* **清單視圖**：表格顯示商品名稱、成交費率、一般日獲利（獲利率%）、活動日獲利。
* **互動**：支援名稱關鍵字搜尋，支援所有欄位點擊排序。
* **單品編輯**：新增/編輯商品資訊（名稱、進價、成交費率、售價、預購開關）。


* **說明網頁 (`/guide`)**：
* 服務條款與隱私政策草案。

## 樣式與響應式要求

1. **Tailwind v4.1**：大量使用 `grid` 與 `flex` 佈局。
2. **卡片設計**：所有計算結果與設定區塊採用圓角卡片 (`rounded-2xl`) 與細微陰影。
3. **深色模式**：確保在 `dark` 類別下，文字顏色為 `slate-200`，背景為 `slate-900`。
4. **手機優化**：表格在手機端自動切換為塊狀顯示或水平滾動。
