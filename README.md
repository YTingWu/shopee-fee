# 蝦皮手續費計算機

一個簡單易用的蝦皮賣家手續費計算工具，幫助您快速計算商品的成本、手續費與獲利。

## 🚀 功能特色

- **即時計算**：輸入數據後即時顯示所有費用明細
- **多種情境**：支援一般日、活動日、免運方案一、免運方案二四種計算模式
- **響應式設計**：完美支援桌面與手機瀏覽
- **直覺介面**：使用 Bootstrap 5 打造美觀且易用的介面

## 📊 計算項目

### 基本手續費
- **成交手續費**：預設 6%（可依商品分類調整）
- **金流手續費**：固定 2.5%

### 特殊情境
- **活動日加收**：額外 +2% 手續費（若參加蝦幣回饋方案則為 0%）
- **較長備貨**：額外 +3% 手續費
- **免運方案一**：總金額加收 6% 手續費
- **免運方案二**：總金額加收 60 元固定手續費
- **蝦幣回饋方案**：
    - 5% 方案：額外 1.5% 活動服務費
    - 10% 方案：額外 3% 活動服務費 (若參加免運活動減免0.5%)

### 稅務設定
- **營業登記**：總金額 1%
- **發票**：總金額 5%
- **蝦皮代開發票**：總金額 5% + 2.5 元固定費用

## 🛠️ 技術架構

- **前端框架**：HTML5 + CSS3 + JavaScript (Vanilla)
- **UI 框架**：Bootstrap 5.3.0
- **圖標**：Bootstrap Icons

## 📱 使用方式

1. 輸入**進價**（商品成本）
2. 輸入**蝦皮價格**（售價）
3. 調整**成交手續費**比例（如需要）
4. 選擇**是否較長備貨**
5. 選擇是否參加**蝦幣回饋方案**
6. 選擇**稅務設定**（如：營業登記、發票等）
7. 查看四種情境下的手續費明細與獲利
