# 蝦皮手續費計算機

一個簡單易用的蝦皮賣家手續費計算工具，幫助您快速計算商品的成本、手續費與獲利。支援**一般賣家**與**商城賣家**兩種賣場模式。

## 🚀 功能特色

- **賣場模式選擇**：支援「一般賣家」與「商城賣家」兩種模式，各有不同的手續費計算規則。
- **雙模式切換**：支援「用售價算獲利」與「用獲利率算售價」兩種模式，滿足不同定價需求。
- **建議售價計算**：輸入預期獲利率，自動反推最合適的建議售價。
- **即時計算**：輸入數據後即時顯示所有費用明細與獲利分析。
- **手續費百分比顯示**：總手續費自動顯示佔售價的百分比，一目了然。
- **多種情境**：支援一般日、促銷檔期、免運方案一、免運方案二等四種情境交叉計算。
- **深色模式**：支援深淺色主題切換，保護您的眼睛。
- **響應式設計**：完美支援桌面與手機瀏覽，隨時隨地都能計算。
- **懸浮資訊欄**：滾動時自動顯示核心數據，方便對照。

## 📊 計算項目

### 賣場模式差異

| 項目 | 一般賣家 | 商城賣家 |
|------|---------|---------|
| 成交手續費來源 | 一般賣家手續費 | 商城賣家手續費 |
| 成交手續費上限 | NT$35,000 | 無上限 |
| 活動日加收（不參加蝦幣回饋） | +2% | +3% |
| 活動日加收（參加蝦幣回饋） | 免收 | 免收 |

### 基本手續費
- **成交手續費**：預設 6%（可依商品分類調整）
- **金流手續費**：固定 2.5%

### 特殊情境
- **促銷檔期加收**：成交手續費額外 +2%（僅在「未參加蝦幣回饋方案」時適用）
- **預購訂單(較長備貨)**：額外 +3% 手續費
- **免運方案一**：總金額加收 6% 手續費
- **免運方案二**：總金額加收 60 元固定手續費
- **蝦幣回饋方案**：
    - 5% 方案：額外 1.5% 活動服務費
    - 10% 方案：額外 2.5% 活動服務費（原為 3%，因同時參加免運活動可折抵 0.5%）
    - **注意**：參加蝦幣回饋方案時，活動日不會額外加收 2% 手續費

### 稅務設定
- **稅籍登記**：總金額 1%
- **發票**：總金額 5%
- **蝦皮代開發票**：總金額 5% + 2.5 元固定費用

## 📱 使用方式

1. 選擇**賣場模式**：
   - **一般賣家**：成交手續費以一般賣家費率計算，成交金額最高計算至 NT$35,000。
   - **商城賣家**：成交手續費以商城賣家費率計算，無成交金額上限。
2. 選擇**計算模式**：
   - **用售價算獲利**：已知售價，想知道扣除手續費後的淨利。
   - **用獲利率算售價**：已知成本，想知道達到目標獲利率該賣多少錢。
3. 輸入**進價**（商品成本）。
4. 根據模式輸入**售價**或**預期獲利率**。
5. 調整**成交手續費**比例（會依據賣場模式顯示不同的費率選項）。
6. 選擇**是否為預購訂單(較長備貨)**（影響 3% 手續費）。
7. 選擇是否參加**蝦幣回饋方案**（5% 或 10%）。
8. 選擇**稅務設定**（如：稅籍登記、發票、蝦皮代開等）。
9. 查看下方四種情境下的手續費明細、建議售價與獲利總覽。
