# 蝦皮手續費計算機

一個簡單易用的蝦皮賣家手續費計算工具，幫助您快速計算商品的成本、手續費與獲利。

## 🚀 功能特色

- **雙模式切換**：支援「用售價算獲利」與「用獲利率算售價」兩種模式，滿足不同定價需求。
- **建議售價計算**：輸入預期獲利率，自動反推最合適的建議售價。
- **即時計算**：輸入數據後即時顯示所有費用明細與獲利分析。
- **多種情境**：支援一般日、促銷檔期、免運方案一、免運方案二等四種情境交叉計算。
- **深色模式**：支援深淺色主題切換，保護您的眼睛。
- **響應式設計**：完美支援桌面與手機瀏覽，隨時隨地都能計算。
- **懸浮資訊欄**：滾動時自動顯示核心數據，方便對照。

## 📊 計算項目

### 基本手續費
- **成交手續費**：預設 6%（可依商品分類調整）
- **金流手續費**：固定 2.5%

### 特殊情境
- **促銷檔期加收**：額外 +2% 手續費（若參加蝦幣回饋方案則為 0%）
- **預購訂單(較長備貨)**：額外 +3% 手續費
- **免運方案一**：總金額加收 6% 手續費
- **免運方案二**：總金額加收 60 元固定手續費
- **蝦幣回饋方案**：
    - 5% 方案：額外 1.5% 活動服務費
    - 10% 方案：額外 3% 活動服務費 (若參加免運活動減免0.5%)

### 稅務設定
- **稅籍登記**：總金額 1%
- **發票**：總金額 5%
- **蝦皮代開發票**：總金額 5% + 2.5 元固定費用

## 📱 使用方式

1. 選擇**計算模式**：
   - **用售價算獲利**：已知售價，想知道扣除手續費後的淨利。
   - **用獲利率算售價**：已知成本，想知道達到目標獲利率該賣多少錢。
2. 輸入**進價**（商品成本）。
3. 根據模式輸入**售價**或**預期獲利率**。
4. 調整**成交手續費**比例（預設 6%，可依分類調整）。
5. 選擇**是否為預購訂單(較長備貨)**（影響 3% 手續費）。
6. 選擇是否參加**蝦幣回饋方案**（5% 或 10%）。
7. 選擇**稅務設定**（如：稅籍登記、發票、蝦皮代開等）。
8. 查看下方四種情境下的手續費明細、建議售價與獲利總覽。
