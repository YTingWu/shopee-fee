## 🚀 功能特色

### 精準計算
- **成交手續費**: 預設 6%，可依商品分類調整
- **金流處理費**: 固定 2.5%
- **免運方案**: 方案一 (6%) / 方案二 (+60元)
- **活動日加收**: 一般賣家 +2% / 商城賣家 +3%
- **預購訂單**: 額外 +3%
- **蝦幣回饋**: 5% 方案 (1.5%) / 10% 方案 (2.5%)

### 稅務支援
- 稅籍登記 (1%)
- 發票 (5%)
- 蝦皮代開發票 (5% + 2.5元)
- 進項稅額扣抵功能

## 📊 計算項目

### 賣場模式差異

| 項目 | 一般賣家 | 商城賣家 |
|------|---------|---------|
| 成交手續費來源 | 一般賣家手續費 | 商城賣家手續費 |
| 成交手續費上限 | NT$35,000 | 無上限 |
| 活動日加收(未參加蝦幣) | +2% | +3% |
| 活動日加收(參加蝦幣) | 免收 | 免收 |

### 基本手續費
- **成交手續費**: 預設 6%（可依商品分類調整）
- **金流手續費**: 固定 2.5%

### 特殊情境
- **促銷檔期加收**: 成交手續費額外 +2% 或 +3%（僅在「未參加蝦幣回饋方案」時適用）
- **預購訂單(較長備貨)**: 額外 +3% 手續費
- **免運方案一**: 總金額加收 6% 手續費
- **免運方案二**: 總金額加收 60 元固定手續費
- **蝦幣回饋方案**:
  - 5% 方案：額外 1.5% 活動服務費
  - 10% 方案：額外 2.5% 活動服務費
  - **注意**: 參加蝦幣回饋方案時，活動日不會額外加收手續費

## 🛠️ 技術架構

- **前端框架**: HTML5 + CSS3 + Vanilla JavaScript
- **樣式框架**: Tailwind CSS v3.4.1
- **圖標庫**: Lucide Icons
- **資料儲存**: localStorage (瀏覽器本地)
- **部署平台**: GitHub Pages

## 📱 使用方式

### 1. 設定賣場資訊
前往「賣場設定」頁面，填寫您的賣家類型、稅務設定等基本資訊。

### 2. 新增商品資料
在「商品管理」頁面新增您的商品，填寫商品名稱、進價、售價、成交手續費率等資訊。

### 3. 快速計算獲利
在「快速計算」頁面從側邊欄選擇商品，或手動輸入數據，即可查看詳細的費用明細與獲利分析。

## 📦 資料備份

所有資料都儲存在您的瀏覽器本地 (localStorage)，不會上傳到任何伺服器。

### 匯出備份
1. 前往「賣場設定」頁面
2. 點擊「匯出資料」按鈕
3. 資料會以 JSON 格式下載

### 匯入備份
1. 前往「賣場設定」頁面
2. 點擊「匯入資料」按鈕
3. 選擇之前匯出的 JSON 檔案
