## 🎯 設計原則

1. **一致性**：所有頁面採用統一的設計語言
2. **可讀性**：大字體、充足留白、清晰層次
3. **專業感**：配色克制、字體精選、動畫流暢
4. **響應式**：所有裝置均有優秀體驗
5. **無障礙**：良好的對比度、focus 狀態明確
6. **功能性**：美化的同時保持所有功能完整

## 🚀 技術特點

- **CSS Variables**：靈活的主題變數系統
- **Modern CSS**：使用 CSS Grid、Flexbox、CSS Variables
- **性能優化**：硬體加速動畫、優化的陰影效果
- **深色模式**：完整支援並自動偵測系統偏好
- **字體優化**：使用 Google Fonts CDN，支援中英文
- **樣式隔離**：專用樣式文件避免衝突
- **樣式覆蓋**：優雅地覆蓋 JS 生成的樣式

## 📱 響應式設計

- **桌面版** (≥768px)：寬敞的佈局，多欄式設計
- **平板版** (≤768px)：適中的間距，堆疊式佈局
- **手機版** (≤640px)：單欄設計，大觸控目標

## 🔧 實現細節

### 樣式覆蓋策略
使用 `!important` 優雅地覆蓋 JavaScript 動態生成的樣式，確保視覺一致性的同時不影響功能。

### CSS Variables 使用
所有顏色、間距、陰影都使用 CSS 變數，易於維護和主題切換。

### 響應式策略
使用 `clamp()` 實現流動字體大小，結合媒體查詢優化不同裝置體驗。

### 動畫性能
使用 `transform` 和 `opacity` 實現動畫，啟用硬體加速，確保 60fps 流暢度。
